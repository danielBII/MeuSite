ALGORITMOS

O algoritmo é nada mais que uma sequência de passos finitos que visam um objetivo. Sendo assim, ele é bastante importante no contexto da computação, uma vez que todas as ações realizadas pelo computador são geradas pelos algoritmos. Tal importância o faz necessário para a aprendizagem de quem quer seguir na área da Informática. Nesse curso, utilizaremos um método de ensino mais descontraído e participativo em relação ao aluno, com a utilização de jogos, posto que é um assunto de baixo rendimento em relação à aprendizagem.

WEB DESIGN

O web design é uma área de estudo que se caracteriza pelo desenvolvimento de interfaces com o usuário em web sites e aplicações web. Podemos considerá-lo como a parte do design gráfico, pois envolve o desenho de qualquer tipo de layout, banner ou mídia computadorizada. O objetivo desse curso é fazer com que o aluno tenha conhecimento de pelo menos o básico das linguagens HTML e CSS.

SISTEMAS OPERACIONAIS

O Sistema Operacional é o software mais importante do computador, pois é ele que inicializa o hardware e promove uma interface amigável entre usuário e máquina, tornando a utilização de um computador mais fácil e agradável. Além disso, ele cria uma plataforma comum a todos os programas utilizados. Abordaremos seus conceitos principais, características, processos e seus sistemas.
